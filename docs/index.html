📖 データの見方

MWEP（月別窓面エネルギー性能）について

  • MWEPh（暖房効果）: 窓からの日射取得による暖房負荷削減量 [MJ/m²]。負値ほど冬の暖房削減効果が大きい。
  • MWEPc（冷房増加）: 窓からの日射による冷房負荷増加量 [MJ/m²]。正値ほど夏の日射遮蔽が重要。
  • MWEPt（正味）: 暖房効果と冷房増加の合計。負値=年間を通して暖房有利、正値=冷房不利。
  • 南面正味収支: 正値の地域（寒冷地）は南窓拡大が有効。負値の地域（温暖地）は庇・ルーバーによる日射遮蔽が重要。
